An expense tracker for all your needs.

Technologies that will be used in this project:

For Frontend:
-> React as framework
-> React-router for navigation
-> Formik+yup for form validation
-> Redux Toolkit for State Management
-> Redux Presistent to store in localStorage
-> React Dropzone for image uploads

For Backend:
-> NodeJS as runtime
-> ExpressJS as backend framework
-> Mongoose for managing our MongoDB
-> jsonwebtoken(JWT) for authentication
-> Multer for file uploading

Currently working on building backend server.
